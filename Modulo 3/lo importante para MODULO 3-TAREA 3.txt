Bootstrap proporciona las clases para que la incrustación sea receptiva. La mayor parte del tiempo que requieren para mostrar 3 rd de aplicaciones de terceros dentro de nuestra página web. Así que podemos usar la <embed>, <iframe>, <video> or <object>etiqueta para incrustar estas aplicaciones de terceros.

Bootstrap proporciona la clase .embed-responsivepara crear incrustaciones responsivas. También podemos proporcionar una relación de aspecto a una relación de aspecto diferente.

.embed-responsive-16by9 - Muestra el video incrustado o en ancho y alto de 16: 9
.embed-responsive-4by3 - Mostrar el contenido insertado o video en ancho y alto de 4: 3

<div class="embed-responsive embed-responsive-4by3">
    <iframe src="https://www.youtube.com/embed/afsdfsdSg5l_Eo" width="425" height="350"></iframe>
</div>
